## Description

Starter for a Nest GraphQL Server w/ TypeScript

## Installation

```bash
yarn install
# or
npm install
```

## Start

```bash
yarn start
# or
npm run start
```

Open url: http://localhost:9000/playground

![](.README/2018-02-19-18-32-07.png)

## Environments

Environments are located in file `.env`.
